# Storyboards pour démos vidéo — ESEBAT

Ces storyboards servent à **enregistrer des vidéos de démonstration** pour chaque profil. Chaque fichier décrit les étapes, la durée indicative et le texte à dire.

## Fichiers disponibles

- **STAFF-creer-demande.md** — Staff : créer et soumettre une demande de matériel

## Comment enregistrer

1. Ouvrir le storyboard correspondant.
2. Lancer l’enregistrement (Loom, OBS, outil intégré, etc.).
3. Suivre les étapes du tableau (écran / action / texte).
4. Publier la vidéo et, si besoin, ajouter le lien dans la plateforme (page Guide utilisateur) ou dans `public/videos/guides/`.

## Idées de storyboards à ajouter

- Point focal : valider une demande, créer une commande, enregistrer une réception.
- Directeur : créer et activer un budget, gérer les utilisateurs.
- Super Admin : configurer un campus et créer un utilisateur.

Vous pouvez dupliquer `STAFF-creer-demande.md` et adapter le contenu pour ces scénarios.
